# AITextDetector

Online demo: https://lyc8503.github.io/AITextDetector/

The model used in this online demo was trained exclusively on Chinese creative writing text. Inputting other text will probably result in systematic bias in the output.

The files `srctrain/train.py` and `srctrain/loader.py` in this repository were written by me. The rest of the code was generated by Claude Opus 4.6 and tested by me.

For more information on this repo, please visit my blog: https://blog.lyc8503.net/post/llm-classifier
